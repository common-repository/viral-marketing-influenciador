== Description ==

= Potencialize suas vendas com marketing de influência! =

O **aplicativo Viral Marketing Influenciadores** é a solução definitiva para alavancar suas vendas através do marketing de influência. Com ele, você pode:

* **Gerar cupons de desconto personalizados** para cada influenciador.
* **Monitorar métricas das vendas** diretamente na plataforma.

= Por que usar a Viral.com.br? =

* **Descobrir influenciadores do seu nicho:** ativos e dos concorrentes.
* **Fácil de usar:** Crie campanhas de sucesso mesmo sem ter experiência prévia.
* **Aumento de vendas:** Potencialize suas vendas com estratégias direcionadas.
* **Análise detalhada:** Acompanhe o desempenho das suas campanhas com métricas precisas.

Transforme sua estratégia de marketing e veja os resultados na sua loja. Comece hoje mesmo com a **Viral**!

= Conexão com viral.com.br =

Você deve conectar sua loja a [viral.com.br](https://viral.com.br) para gerar automaticamente cupons de desconto personalizados para cada influenciador, além de receber métricas de vendas dos perídos das suas campanhas com influenciadores. A conexão dá a Viral permissão para ler/escrever cupons de desconto e também para ler pedidos. Se deseja saber mais sobre como a Viral utiliza dados coletados, acesse nossa [Política de Privacidade](https://viral.com.br/politica-de-privacidade) e nossos [Termos de Serviço](https://viral.com.br/termos-de-servico)

== Installation ==

= Requisitos mínimos =

* PHP 7.4 or maior é necessário (PHP 8.0 or maior é recomendado)

= Instalação automática =

Instalação automática é a opção mais fácil -- O WordPress irá cuidar de tudo, e você não precisará sair do seu navegador. Para fazer uma instalação automática do plugin Viral, faça login no seu painel WordPress e navegue até o menu Plugins, clique em "Adicionar plugin".

No campo de busca procure "Viral", aperte enter. Quando você encontrar o plugin, você pode instalar o plugin clicando em "Instalar agora", o WordPress fará o restante.

= Instalação Manual =

Instalação manual requer baixar o plugin Viral e fazer o upload no seu web server usando sua aplicação favorita de FTP. O WordPress contem [instruções de como fazer isso aqui](https://wordpress.org/support/article/managing-plugins/#manual-plugin-installation).

== Frequently Asked Questions ==

= O que acontece após a instalação? =

Você será redirecionado para a https://viral.com.br, onde o processo para conectar ao seu woocommerce se iniciará automaticamente. Depois, você será redirecionado para a tela de confirmação da sua loja dando permissão para criarmos e lermos os cupons da sua loja, além da leitura dos pedidos para criarmos métricas de venda e acompanhar o desempenho dos influenciadores.
